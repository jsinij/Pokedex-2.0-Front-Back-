# Pokédex TypeScript

Una aplicación full-stack moderna para explorar, crear y gestionar Pokémon personalizados. Construida con React, TypeScript, Express.js y Prisma.

## Características

- **Exploración de Pokémon**: Navega por la Pokédex oficial y Pokémon personalizados
- **Crear Pokémon Custom**: Los administradores pueden crear Pokémon personalizados con tipos, sprites y evoluciones
- **Sistema de Usuarios**: Registro, login y gestión de roles (usuario/admin)
- **Interfaz Game Boy**: Diseño retro inspirado en la Pokédex clásica
- **Evoluciones**: Define cadenas de evolución para tus Pokémon personalizados
- **Autenticación JWT**: Sistema seguro de login con tokens JWT

## Tecnologías

### Frontend
- **React 18** con TypeScript
- **Vite** como bundler
- **CSS3** para estilos (responsive design)
- **Hooks** para manejo de estado

### Backend
- **Node.js** + **Express.js**
- **TypeScript** para type safety
- **Prisma ORM** para gestión de base de datos
- **SQLite** como base de datos
- **bcrypt** para hash de contraseñas
- **JWT** para autenticación
